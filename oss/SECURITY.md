# Security Policy

Veyl welcomes security review of the public client source and the behavior of the hosted service.

Do not access, modify, delete, disrupt, or attempt to exfiltrate data that does not belong to accounts you control. Do not run automated load, spam, scraping, credential-stuffing, or denial-of-service tests against Glyphteck-managed infrastructure without written permission.

Good reports include the affected surface, steps to reproduce, expected impact, and a minimal proof using accounts and data you control.

Report sensitive issues privately to contact@glyphteck.com.
